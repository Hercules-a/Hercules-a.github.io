VR Game in Unity:

Game is creating in Unity Game Engine, for VR Occulus Quest headset.
Currently in the game is implemented:
  • player movement in continuous and teleportation mode
  • basic player interactions with objects
  • simple physics
  • rules of billiards
Project's code on GitHub
We can choose from three modes:
  • practice:
    • each ball can be picked up and moved to any position on the table
    • you can hit any ball
    • you can restart the game at any time
    • no rules of the game
  • one device duel:
    • allows 2 people to play alternately on one device
    • the game guards the rules
    • a player can only pick up the cue ball after an opponent's foul
    • it is only possible to hit the cue ball when the balls have stopped
    • in order to avoid the accidental hitting of the cue ball,
      the collision between the cue stick and the cue ball is activated by pressing the button
    • displays information about whose movement it is and whether a foul has been committed
  • online:
    • connecting players using PUN Photonengine
    • players can connect with friends using a generated code
    • players can connect to random players
